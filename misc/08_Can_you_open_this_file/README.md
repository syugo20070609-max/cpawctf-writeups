# Can you open this file ?

## カテゴリ
Misc

## どんな問題だったか
拡張子のないファイルを渡されて、中身を確認してフラグを取得する問題。

## アプローチ
1. `file open_me` でファイルの種類を調べたらWordファイル(.doc)とわかった
2. 拡張子 `.doc` をつけてリネームした
3. Wordで開いてフラグを取得した

## 詰まったところ
拡張子がないと何のファイルかわからないが、`file`コマンドで種類を特定できることを応用した。
(exec_meで学んだことを活かせた)

## 学んだこと
- `file`コマンドは実行ファイル以外にも使える
- 拡張子とファイルの実際の種類は別物G
