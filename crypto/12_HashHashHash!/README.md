# HashHashHash!

## カテゴリ
Crypto

## どんな問題だったか
SHA1ハッシュ値を元の値に戻す問題。

## アプローチ
1. ハッシュ値をそのままGoogleで検索した
2. レインボーテーブルサイトに元の値が載っていた

## 学んだこと
- ハッシュ関数は一方向性(元に戻せない)が特徴
- ただしSHA1やMD5は既知のハッシュ値がレインボーテーブルとして公開されている
- 短い・単純な文字列のハッシュはGoogle検索だけで逆引きできてしまう
- パスワード保存にSHA1をそのまま使うのは危険
