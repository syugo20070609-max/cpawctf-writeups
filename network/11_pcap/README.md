# pcap

## カテゴリ
Network

## どんな問題だったか
pcapファイル(ネットワークのパケットを記録したファイル)を解析してフラグを取得する問題。

## アプローチ
1. WiresharkでpcapファイルをGUIで開いた
2. パケットの詳細パネル(右下)の16進数ダンプを見た
3. 生データの中にフラグが埋め込まれているのを発見した

## 詰まったところ
パケットが2つしかなく、どこを見ればいいか最初わからなかった。
16進数の右側にASCII文字列として表示されているところにフラグがあった。

## 学んだこと
- pcapファイルはWiresharkで開ける
- パケットの生データ(16進数ダンプ)にASCII文字列が含まれることがある
- ネットワークを流れるデータは平文で見えることがある
