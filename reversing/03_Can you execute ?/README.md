# exec_me

## カテゴリ
Reversing

## どんな問題だったか
拡張子のないファイルを渡されて、実行してフラグを取得する問題。

## アプローチ
1. `file exec_me` でファイルの種類を調べたらELFバイナリ(Linux実行ファイル)とわかった
2. `chmod +x exec_me` で実行権限を付与した
3. `./exec_me` で実行してフラグを取得した

## 詰まったところ
拡張子がないとWindowsでは実行できない。
LinuxではELFバイナリであることを`file`コマンドで確認する必要があった。

## 学んだこと
- `file`コマンドでファイルの種類を調べられる
- ELFはLinuxの実行ファイル形式
- `chmod +x`で実行権限を付与できる
